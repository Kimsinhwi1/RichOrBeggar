## 요약 설명 (Short description, 최대 132자)
```
쿠팡 주문내역을 클릭 한 번으로 엑셀(CSV)로 추출하고 월별·카테고리별 지출을 분석. 서버 전송 없이 내 브라우저에만 저장하는 쿠팡 가계부.
```

## 상세 설명 (Detailed description)
```
쿠팡 주문내역을 엑셀로 한 번에 내려받고, 한 해 지출을 자동으로 정리하세요.

■ 이런 분께
· 쿠팡 지출을 가계부에 정리하고 싶은 분
· 법인카드/사업용 쿠팡 구매를 경비 정산해야 하는 직장인·자영업자
· 쿠팡에서 한 달에 얼마 쓰는지 알고 싶은 분

■ 주요 기능 (무료)
· 주문내역 추출 — 주문목록 페이지에서 클릭 한 번, 여러 페이지 자동 순회
· 엑셀(CSV) 다운로드 — 날짜·주문번호·상품명·수량·금액·배송비·상태
· 주문번호 합산 뷰 — 같은 주문을 묶어 총액 표시 (카드명세서 대조에 딱)
· 지출 대시보드 — 총 지출, 주문 수, 상품 수 한눈에

■ 개인정보 보호 (가장 중요)
· 주문 데이터는 오직 내 브라우저(로컬)에만 저장됩니다.
· 어떤 데이터도 외부 서버로 전송하지 않습니다.
· 쿠팡 계정에 아무것도 쓰지 않습니다 (읽기 전용).
· 별도 회원가입이 필요 없습니다.

■ 사용 방법
1. 쿠팡 로그인 후 "주문목록" 페이지로 이동
2. 화면 우하단 "내역 가져오기" 버튼 클릭
3. 대시보드에서 지출 확인 및 CSV 다운로드

■ 필요 권한
· coupang.com 접근 — 주문목록 페이지를 읽어 내역을 추출하기 위해서만 사용
· 저장소 — 추출한 내역을 내 브라우저에 저장

문의: th280140@gmail.com
```
