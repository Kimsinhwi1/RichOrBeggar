# 개인정보 처리방침 — 쿠팡 가계부

_최종 업데이트: 2026-07-02_

쿠팡 가계부(이하 "본 확장")는 사용자의 개인정보를 최우선으로 보호합니다.

## 1. 수집하는 정보
본 확장은 **어떠한 개인정보도 외부로 수집·전송하지 않습니다.**

- 본 확장은 사용자가 직접 "내역 가져오기"를 실행할 때, 사용자가 로그인한 쿠팡 주문목록 페이지에 표시된 주문 정보(주문일, 주문번호, 상품명, 수량, 금액, 배송 상태)를 읽습니다.
- 읽어들인 데이터는 **사용자의 브라우저 내 저장소(IndexedDB)에만** 저장됩니다.
- 이 데이터는 사용자의 컴퓨터를 벗어나지 않으며, 개발자를 포함한 제3자에게 전송되거나 공유되지 않습니다.

## 2. 데이터 사용 목적
읽어들인 주문 데이터는 오직 사용자 본인에게 지출 리포트와 엑셀(CSV) 내보내기를 제공하기 위한 목적으로만, 사용자의 브라우저 안에서 처리됩니다.

## 3. 데이터 저장 및 보관
- 모든 데이터는 사용자 브라우저의 로컬 저장소에 보관됩니다.
- 사용자는 언제든지 확장 삭제 또는 브라우저 데이터 삭제를 통해 저장된 데이터를 완전히 제거할 수 있습니다.

## 4. 권한 사용 안내
- **coupang.com 접근 권한**: 주문목록 페이지에서 주문 내역을 읽기 위해서만 사용합니다. 다른 사이트에는 접근하지 않습니다.
- **저장소(storage) 권한**: 추출한 내역과 설정을 사용자 브라우저에 저장하기 위해 사용합니다.
- 본 확장은 쿠팡 계정에 어떠한 쓰기 동작(주문/취소 등)도 수행하지 않습니다. **읽기 전용**입니다.

## 5. 제3자 제공
없음. 본 확장은 데이터를 제3자에게 제공하거나 판매하지 않습니다.
