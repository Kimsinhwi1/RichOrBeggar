# 크롬 웹스토어 "개인정보 보호 관행" 탭 입력 가이드 (복사용)

스토어 심사에서 요구하는 항목별로 아래 문구를 그대로 붙여넣으면 됩니다.

---

## 1. 단일 목적 (Single purpose)
```
이 확장 프로그램의 단일 목적은, 사용자가 로그인한 쿠팡 주문목록 페이지에서 본인의 주문내역을 추출하여 지출을 CSV로 내보내고 대시보드로 확인하도록 돕는 것입니다.
```

## 2. storage 권한 사유
```
추출한 주문내역과 사용자 설정을 사용자 브라우저의 로컬 저장소(IndexedDB, chrome.storage)에 저장하기 위해 필요합니다. 저장된 데이터는 외부 서버로 전송되지 않으며 사용자 기기에만 보관됩니다.
```

## 3. 호스트 권한 사유 (mc.coupang.com)
```
사용자가 로그인한 쿠팡 주문목록 페이지(https://mc.coupang.com)에서 화면에 표시된 주문내역을 읽어 추출하기 위해 필요합니다. 해당 도메인 외 다른 사이트에는 접근하지 않습니다.
```

## 4. 원격 코드 사용 (Remote code)
- **선택: "아니요, 원격 코드를 사용하지 않습니다."**
- 이유: 본 확장은 원격에서 실행 가능한 코드(JS/Wasm 등)를 불러오지 않습니다.
  화면 구조 변경에 대비해 GitHub의 공개 설정 파일(`selectors.json`)을 내려받지만,
  이는 **실행 코드가 아니라 JSON 설정 데이터**이며 파싱에만 사용됩니다. (MV3에서 원격 '데이터'는 허용)
- 혹시 심사자가 network 요청에 대해 물으면: "쿠팡 페이지 구조 변경 시 재배포 없이 복구하기 위한
  셀렉터 설정(JSON 데이터)을 내려받는 것이며, 코드 실행이 아닙니다."라고 답하면 됩니다.

## 5. 데이터 사용 (Data usage) — 인증 체크박스
"데이터 수집" 항목에서: 본 확장은 어떤 사용자 데이터도 **외부로 전송·수집하지 않으므로**,
나열된 데이터 유형(개인정보, 위치, 금융정보 등) 중 **아무것도 선택하지 않습니다(수집 안 함).**

그리고 하단 인증 3가지에 모두 체크:
- [x] 승인된 용도 외 제3자에게 데이터를 판매/양도하지 않습니다.
- [x] 항목의 단일 목적과 무관한 용도로 데이터를 사용/양도하지 않습니다.
- [x] 신용도 판단 또는 대출 목적으로 데이터를 사용/양도하지 않습니다.

개발자 대시보드 **설정(Account) → 연락처 이메일**에 아래 주소 입력 후, 받은 인증 메일의 링크로 **인증 완료**:
```
th280140@gmail.com
```
> 이 이메일 인증이 안 되면 게시가 막힙니다. 항목 저장 전에 먼저 인증하세요.
